# Simple VueJS ToDo Websocket Server.

## Requirements;
- NodeJS Version 13.9.0 (Use NVM / Node Version Manager to handle multiple node versions)
- WS plugin.
- Sqlite3 plugin.

## Installation;
run `npm install`

## Running;
run `npm start`

## Notes;
This server is programmed to listen on port 9001. Ensure this port is available.
